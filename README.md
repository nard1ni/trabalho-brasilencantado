 🌿 Brasil Encantado — Portal de Turismo

 Descrição do Projeto

**Brasil Encantado** é um portal de turismo fictício desenvolvido como projeto acadêmico, com o objetivo de apresentar os principais destinos turísticos, gastronomia e cultura do Brasil. O site foi construído com foco em HTML semântico, CSS externo, responsividade e boas práticas de desenvolvimento web.

---

 🗂️ Estrutura do Projeto

```
/projeto-turismo
│
├── index.html          → Página inicial com hero, destinos em destaque e dicas
├── destinos.html       → Destinos por região (Nordeste, Sudeste, Sul, Norte, CO)
├── gastronomia.html    → Culinária regional, pratos típicos e galeria
├── contato.html        → Formulário de contato e canais de atendimento
│
├── css/
│   └── style.css       → Folha de estilos externa (único arquivo CSS)
│
├── img/
│   └── (pasta reservada para imagens locais)
│
└── README.md


 HTML Semântico
-  <header> em todas as páginas
-  <nav> com menu de navegação e link ativo
-  <main> envolvendo o conteúdo principal
-  <section> para cada bloco de conteúdo
-  <aside> com curiosidades, dicas e informações extras
-  <footer> completo em todas as páginas
-  <article> para cards de destinos e pratos
-  <nav aria-label="..."> para breadcrumbs e menu principal

 CSS Externo (`css/style.css`)
-  Arquivo CSS único e externo
-  Variáveis CSS (`:root`) para cores e tipografia
-  Paleta de cores coerente (verde, amarelo, azul)
-  Google Fonts: Playfair Display + Nunito
-  Flexbox para todos os layouts
-  Espaçamentos consistentes
-  Responsividade com `@media` queries
-  Sem CSS inline

 Imagens
-  6+ imagens com atributo `alt` descritivo
-  Imagens via URL externo (Unsplash)
-  Pasta `/img` disponível para imagens locais

Vídeos do YouTube
-  4 vídeos incorporados com `<iframe>`
-  Vídeos relacionados ao turismo e cultura do Brasil

 Aside
-  Curiosidades históricas e geográficas
-  Dicas práticas de viagem
-  FAQ e informações de contato

 Footer

-  Ano: 2026
-  Contato fictício completo
